---
term: RDBMS
short: Relational database management system — the software that runs a relational database (SQLite, PostgreSQL…).
---

RDBMS stands for *relational database management system*: a [[dbms]] built on the
[[relational-model]]. SQLite, PostgreSQL, MySQL, Oracle, and SQL Server are all
RDBMSs.

Through the 1980s and 90s the RDBMS became simply *the* database — the assumed
substrate of business software — a position it still largely holds.
