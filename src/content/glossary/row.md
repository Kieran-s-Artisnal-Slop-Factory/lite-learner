---
term: Row
short: A single record in a table — one complete entry, with a value for each column.
---

A row is one entry in a [[table]]: one contact, one book, one order. Every row
carries a value for each [[column]] the table defines, and in most tables a
[[primary-key]] gives each row a unique identifier so it can be referred to
unambiguously.

If a table is a grid, rows are its horizontal lines — each one a single *thing*
the table tracks.
