---
term: Column
short: A named field in a table — one fact, with a fixed type, that every row carries.
---

A column is one named fact that every [[row]] in a [[table]] has — `title`,
`year`, `email`. Each column has a [[data-type]] that constrains what it can
hold, so a number column rejects a value like "next Tuesday".

If a table is a grid, columns are its headings: they describe *what* is being
recorded, while rows are the entries.
