[Microsoft SQL Server](https://www.microsoft.com/en/sql-server) is Microsoft's flagship relational database management system ([[relational-database|RDBMS]]). Like [[postgresql|PostgreSQL]] and [[mysql|MySQL]] it runs as a dedicated server that applications connect to with [[sql|SQL]], and it competes with [[oracle|Oracle]] and [[db2|Db2]] in the enterprise market.

## Features & Architecture

SQL Server speaks a SQL dialect called **T-SQL** (Transact-SQL), which adds procedural programming, and ships with a rich tooling ecosystem: SQL Server Management Studio for administration, Integration Services for data pipelines, and Analysis/Reporting Services for business intelligence. It is tightly integrated with Windows, Active Directory [[access-control|authentication]], and the .NET development stack, which made it the default choice in Microsoft-centric organizations. Since 2017 it also runs on Linux and in containers.

## History

SQL Server began in 1989 as a port of **Sybase SQL Server** to OS/2 — making it, via Sybase, a descendant of Berkeley's [[ingres|Ingres]] lineage. Microsoft and Sybase parted ways in the mid-1990s, and Microsoft rewrote and grew the product into a full enterprise platform. It remains one of the most widely deployed commercial databases, especially in corporate and government environments.
