---
term: Relational database
short: A database that organizes data as tables connected by ids, with enforced rules. The most common kind.
---

A relational database stores data as [[table|tables]] of [[row|rows]] and
[[column|columns]], linked together by ids ([[relationship|relationships]]) and
governed by rules that keep the data consistent. It's the default,
general-purpose choice — SQLite, PostgreSQL, and MySQL are all relational — and
you query it with [[sql]].

The model comes from Edgar Codd's [[relational-model]], and the software that
runs one is a relational database management system, or [[rdbms]].
