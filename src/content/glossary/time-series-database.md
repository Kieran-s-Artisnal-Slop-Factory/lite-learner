---
term: Time-series database
short: A database optimized for streams of timestamped measurements — metrics, sensors, stock ticks.
---

A time-series database is built for endless streams of timestamped measurements:
server temperatures, heart rates, stock ticks.

It's optimized to append constantly, query by time range, and summarize per
minute/hour/day, expiring or compressing old data automatically. Examples:
InfluxDB, TimescaleDB.
