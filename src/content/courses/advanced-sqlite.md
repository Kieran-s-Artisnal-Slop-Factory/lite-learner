Go under the hood. This course covers the SQLite features that power production
systems — and how the database works internally. You'll configure it with
PRAGMAs, write recursive CTEs and reusable views, automate logic with triggers,
search text and space with the built-in extensions, crunch numbers with window
functions, and tune the query planner.

It ends with a capstone that builds a production-style bookstore database using
nearly everything you've learned. This course assumes the Beginner and
Intermediate material — queries and joins, constraints, transactions, and
indexes.
