The first thing you do with an unfamiliar database is ask what's in it. Every
table is recorded in `sqlite_master`, so listing them is a `SELECT` away:

```sql
SELECT name FROM sqlite_master WHERE type = 'table';
```

The `WHERE type = 'table'` keeps indexes and views out of the list, leaving just
the tables. (In a real SQLite shell you'd type `.tables`, but that's a shell
shortcut — under the hood it reads `sqlite_master` exactly like this.)

## Your task

Save the list of table names into a new table called `answer`:

```sql
CREATE TABLE answer AS
SELECT name FROM sqlite_master WHERE type = 'table';
```

Check your solution — this database has a `directors` table and a `movies`
table.
